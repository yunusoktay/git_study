# Day 002

## Focus

- Learn `if/else`
- Practice `for` loops
- Understand `while` and `do-while`
- Learn `switch`

## Run

From the repository root:

```bash
dart run dart/day_002.dart
```

## Outcome

Day 2 is about control flow. Control flow helps a program make decisions and repeat actions.

`if/else` is used for simple decisions. It runs different code when a condition is true or false.

`for` loops are useful when you want to repeat code for a known number of times.

`while` and `do-while` loops repeat code while a condition is true.

`do-while` is different because it runs the code one time before checking the condition.

`switch` is useful when one variable has many possible values. It can be cleaner than a long `if/else if` chain.
